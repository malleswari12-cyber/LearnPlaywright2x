# VS Code Keyboard Shortcuts — macOS

| Action | Shortcut | Description |
|---|---|---|
| Command Palette | `Cmd + Shift + P` | Open the command palette |
| Open File | `Cmd + O` | Open a file |
| New File | `Cmd + N` | Create a new file |
| Save | `Cmd + S` | Save the current file |
| Save All | `Cmd + Option + S` | Save all open files |
| Close Editor | `Cmd + W` | Close the current editor tab |
| Split Editor | `Cmd + \` | Split the editor view |
| Toggle Terminal | `Ctrl + ` `` | Show or hide integrated terminal |
| Find | `Cmd + F` | Find in current file |
| Replace | `Cmd + Option + F` | Replace in current file |
| Find in Files | `Cmd + Shift + F` | Search across workspace |
| Replace in Files | `Cmd + Shift + H` | Replace across workspace |
| Go to Line | `Ctrl + G` | Jump to a line number |
| Go to File | `Cmd + P` | Open file by name |
| Comment Line | `Cmd + /` | Toggle line comment |
| Format Document | `Shift + Option + F` | Format entire file |
| Rename Symbol | `F2` | Rename symbol across workspace |
| Show Problems | `Cmd + Shift + M` | Open Problems panel |
| Toggle Sidebar | `Cmd + B` | Show or hide sidebar |
| Open Settings | `Cmd + ,` | Open VS Code settings |
| Zen Mode | `Cmd + K Cmd + Z` | Enter distraction-free mode |
| Toggle Full Screen | `Ctrl + Cmd + F` | Enter full screen mode |
| Open Extensions | `Cmd + Shift + X` | Open extensions view |
| Run Task | `Cmd + Shift + B` | Run a build task |
| Peek Definition | `Option + F12` | Peek definition inline |
