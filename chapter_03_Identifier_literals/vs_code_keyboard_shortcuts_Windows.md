# VS Code Keyboard Shortcuts — Windows

| Action | Shortcut | Description |
|---|---|---|
| Command Palette | `Ctrl + Shift + P` | Open the command palette |
| Open File | `Ctrl + O` | Open a file |
| New File | `Ctrl + N` | Create a new file |
| Save | `Ctrl + S` | Save the current file |
| Save All | `Ctrl + K S` | Save all open files |
| Close Editor | `Ctrl + W` | Close the current editor tab |
| Split Editor | `Ctrl + \` | Split the editor view |
| Toggle Terminal | `Ctrl + ` `` | Show or hide integrated terminal |
| Find | `Ctrl + F` | Find in current file |
| Replace | `Ctrl + H` | Replace in current file |
| Find in Files | `Ctrl + Shift + F` | Search across workspace |
| Replace in Files | `Ctrl + Shift + H` | Replace across workspace |
| Go to Line | `Ctrl + G` | Jump to a line number |
| Go to File | `Ctrl + P` | Open file by name |
| Comment Line | `Ctrl + /` | Toggle line comment |
| Format Document | `Shift + Alt + F` | Format entire file |
| Rename Symbol | `F2` | Rename symbol across workspace |
| Show Problems | `Ctrl + Shift + M` | Open Problems panel |
| Toggle Sidebar | `Ctrl + B` | Show or hide sidebar |
| Open Settings | `Ctrl + ,` | Open VS Code settings |
| Zen Mode | `Ctrl + K Z` | Enter distraction-free mode |
| Toggle Full Screen | `F11` | Enter full screen mode |
| Open Extensions | `Ctrl + Shift + X` | Open extensions view |
| Run Task | `Ctrl + Shift + B` | Run a build task |
| Peek Definition | `Alt + F12` | Peek definition inline |
